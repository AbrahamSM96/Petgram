# Petgram

This is a PWA mobile first that is a little same an Instagram.
You can create users, sign in, like photos and you can see your favorites photos.
The tecnologies uses in this project:
*ReactJS
*Hooks
*Graphql
*Webpack
*Cypress
*Now
*Context/api
*React/router
